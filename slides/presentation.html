Step 2

Cloud Chat — FirebaseAIService

import 'package:firebase_ai/firebase_ai.dart'; class FirebaseAIService implements AIService { GenerativeModel? _model; Future<void> initialize() async { final ai = FirebaseAI.googleAI(); _model = ai.generativeModel(model: 'gemini-2.5-flash'); } Stream<String> generateResponseStream(String prompt) async* { final stream = _model!.generateContentStream([Content.text(prompt)]); await for (final chunk in stream) { if (chunk.text != null) yield chunk.text!; } } }

Step 4

Local Inference — LocalAIService

FlutterGemma.initialize(); // Download model (~500 MB) await FlutterGemma.installModel(modelType: ModelType.gemmaIt) .fromNetwork(modelUrl, token: hfToken) .withProgress((p) => onProgress(p / 100)).install(); // Create model + chat session final model = await FlutterGemma.getActiveModel( maxTokens: 1024, preferredBackend: PreferredBackend.gpu); _chat = await model.createChat(); // Generate response await _chat.addQuery(Message.text(text: prompt, isUser: true)); await for (final chunk in _chat.generateChatResponseAsync()) { if (chunk is TextResponse) yield chunk.token; }

 isUser: true is critical! Default is false → empty responses

Step 5

Hybrid Strategy — HybridAIService

4 Strategies

localFirst

Local → Cloud

cloudFirst

Cloud → Local

localOnly

Offline mode

cloudOnly

Cloud only

Fallback Pattern

case AIStrategy.localFirst: try { yield* local.generateResponseStream(prompt); } catch (e) { yield* cloud.generateResponseStream(prompt); }

UI: 4-option SegmentedButton

Cloud | Local | Local+Cloud | Cloud+Local — yield* forwards the entire stream, try/catch switches to fallback

Step 6

Embeddings + VectorStore

Install & Load

// Install embedding model await FlutterGemma.installEmbedder() .modelFromNetwork(modelUrl, token: hfToken) .tokenizerFromNetwork(tokenizerUrl, token: hfToken) .install(); // Initialize VectorStore (SQLite) await FlutterGemmaPlugin.instance .initializeVectorStore('rag.db'); // Load docs (auto-embeds!) await FlutterGemmaPlugin.instance.addDocument( id: 'paris', content: description, metadata: jsonEncode({'city': 'Paris'}), );

How Embeddings Work

"cat"  →  [0.2, 0.8, 0.1, ...]

"kitten" →  [0.3, 0.7, 0.2, ...] ~ similar!

"car"   →  [0.9, 0.1, 0.7, ...] ~ different
